LearnNow Website Test Plan

**Objective:** Each feature will have the intended usability,
functionality, and security to make the website meet its requirements
and purpose

**Scope:** All features will be tested

- Pages load and display properly in browser and sign up and login works

- data remembrance such as username, learning path selected, subject
  selected user progress saved, and user progress displayed

- All buttons and functions on course page work as intended and all
  games work as intended

- chat functions to allow message sending between users

- available settings allow users to customize experience

- user personal data secure and unauthorized access to pages prohibited

**Environment:** Tests will be conducted on local computer in a browser

**Resources:**

I will test the core functionality of the website and students in grades
1-6 will test its usability and effectiveness then give feedback

The students will detail the website's usability and how likely they
would be to use it for entertainment and learning purposes

The students will also detail how engaging the games are and how
effective non-game mode is at helping them practice what they learned

Questions to Ask:

Is this website easy to use and move around on?

What is one thing you would change or add to make this website better?

Do you think the game is fun and helps you practice what you learned in
school ?

Do you think this website can help you practice what you learned in
school? If not, how can it be better?

Do you like seeing your score right away and seeing other people's
scores?

Do you like being able to chat with other users and change the screen
from light to dark?

On a scale from 1 to 10, how likely are you to use this website on your
own for learning or fun?

On a scale from 1 to 10, how likely are you to use this website if your
friends were using it too?
