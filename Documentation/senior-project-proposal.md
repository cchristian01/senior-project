Problem Statement: Learning websites are important because people always
have a desire to learn but sometimes the learning environment is dull,
and this dims the person's desire to continue. The goal of my website is
to keep the learning environment interactive and competitive. This will
make it entertaining and create an environment where the users want to
continue to develop and learn.

In an ideal state the website is fully functional with at least one
subject or topic covered that allows the user to effectively progress
through the course while gaining valuable knowledge on the topic.
Entertainment is also brought to the learning environment using mini
games. The project will be completed with only one or very few topics
covered, and few games embedded in its initial design. However, the
project will be allowed to expand and develop through time a be able to
support more topics and more users.

Project Description: A website designed for people to learn in a fun
interactive way by using mini games and a strategic and effective
progression of learning to generate a real understanding of the topics
and ideas discussed for all users

Proposed Implementation Languages: HTML, CSS, MySQL, JAVASCRIPT, C#

Additional Software/Equipment Needed: node js, tailwindcss, react
library for javascript and typescript, unity for game development

Personal Motivation: This project will develop my knowledge of web
development by incorporating the software technologies learned
throughout the degree to build a practical and fully functional easy to
use website. It will specifically incorporate the use of databases
demonstrating knowledge of backend and front-end software engineering
capabilities as well as upholding any necessary security measures.

Outline of Future Research Efforts: The future goals of this project
will be to expand the features from their initial state. This includes
adding more learning games and subjects as well as expanding its
functionality and design to be able to support more users.
